- 📁 File & image sharing with **live upload/download progress indicators**

### 🛒 E-Commerce — Full-Stack Shopping Platform

> A complete 3-part e-commerce system: customer storefront, admin dashboard and REST API backend — all deployed and live.

[![Live Demo](https://img.shields.io/badge/🔗_Live_Demo-e85d04?style=for-the-badge)](https://e-commerce-frontend-five-khaki.vercel.app/)
[![Frontend](https://img.shields.io/badge/GitHub-Frontend-181717?style=for-the-badge&logo=github)](https://github.com/Harsh-Kumar-Pandit/E-commerce-frontend)
[![Backend](https://img.shields.io/badge/GitHub-Backend-181717?style=for-the-badge&logo=github)](https://github.com/Harsh-Kumar-Pandit/E-commerce-backend)
[![Admin](https://img.shields.io/badge/GitHub-Admin_Panel-181717?style=for-the-badge&logo=github)](https://github.com/Harsh-Kumar-Pandit/Ecommerce-admin)

**What I built:**

🛍️ **Customer Storefront**
- Product catalog with **filtering, sorting and search**
- Shopping cart with quantity management and order summary
- Secure **user auth** with login/register flow

🖥️ **Admin Dashboard**
- Product management — add, edit, delete listings with **Cloudinary image uploads**
- Order management with **status tracking and fulfillment**
- Revenue stats with **charts and time-range filters**
- Secure admin login with token management

⚙️ **Backend API**
- Full **REST API** with controllers, routes, middleware and validation layers
- **Stripe & Razorpay** payment gateway integration
- **Cloudinary** for image/video management
- Input validation, error handling and CORS setup

| Layer | Tech |
|---|---|
| Frontend | React.js, Tailwind CSS, CSS, React Router |
| Admin Panel | React.js, Tailwind CSS, Vite, React Toastify |
| Backend | Node.js, Express.js, MongoDB, Mongoose |
| Payments | Stripe |
| Media | Cloudinary |
| Auth | JWT, bcrypt |
